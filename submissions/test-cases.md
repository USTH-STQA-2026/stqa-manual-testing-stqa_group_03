## Bước 1: Mô hình hóa miền đầu vào — Input Domain Modeling (IDM)

> 📖 **Textbook:** Chương 6 — _Input Domain Modeling_, Paul Ammann & Jeff Offutt.
>
> **Trước khi viết Test Case**, nhóm **phải** phân tích miền đầu vào bằng bảng IDM bên dưới.
> Mỗi chức năng cần xác định: **Đặc tính (Characteristic)**, **Phân vùng (Block/Partition)**, và **Giá trị đại diện (Value)**.

| -------------------------- | ----------------- | ------------------------ | ---------------------------- |
| Email có tồn tại trong DB? | Có                | `librarian@library.com`  | Đăng nhập thành công         |
|                            | Không             | `noone@email.com`        | Thông báo lỗi                |

### IDM — Tìm kiếm & Lọc sách (REQ-03)

| Đặc tính (Characteristic) | Phân vùng (Block) | Giá trị đại diện (Value)                     | Kết quả mong đợi                                                                 |
| ------------------------- | ----------------- | -------------------------------------------- | -------------------------------------------------------------------------------- |
| Nhập ký tự tìm kiếm       | Viết thường       | `"flutter"`                                  | Trả về kết quả chính xác không phân biệt hoa thường.                             |
|                           | Viết hoa          | `"FLUTTER"`                                  | Trả về kết quả chính xác không phân biệt hoa thường.                             |
| Định dạng chuỗi ký tự     | Chứa khoảng trắng | `" Flutter "`                                | Hệ thống tự động xóa khoảng trắng thừa (trim space) và tìm đúng sách.            |
| Logic bộ lọc Thể loại     | Chọn danh mục     | Thể loại: `"Công nghệ"`                      | Danh sách hiển thị thu hẹp, chỉ chứa các sách thuộc đúng thể loại được chọn.     |
| Kết hợp Tìm kiếm & Lọc    | Điều kiện AND     | Thể loại: `"Kinh tế"` + Từ khóa: `"Flutter"` | Trả về danh sách trống và hiển thị thông báo "Không tìm thấy sách".              |
| Khôi phục danh sách       | Hủy bỏ bộ lọc/tìm | Thực hiện xóa/reset                          | Giao diện xóa từ khóa và bộ lọc, tải lại toàn bộ danh sách sách mặc định đầy đủ. |

### IDM — Trả sách (REQ-05)

| Đặc tính (Characteristic) | Phân vùng (Block) | Giá trị đại diện (Value) | Kết quả mong đợi                                |
| ------------------------- | ----------------- | ------------------------ | ----------------------------------------------- |
| Trạng thái đăng nhập      | Đã đăng nhập      | Tài khoản thành viên     | Cho phép truy cập và thực hiện trả sách         |
|                           | Chưa đăng nhập    | Không có session         | Chuyển hướng về trang đăng nhập                 |
| Thời hạn phiếu mượn       | Đúng hạn          | Phiếu mượn còn hạn       | Trả sách thành công, không hiển thị cảnh báo    |
|                           | Quá hạn           | Phiếu mượn hết hạn       | Trả sách thành công, kèm cảnh báo quá hạn       |
| Trạng thái nút bấm        | Chưa trả sách     | Phiếu mượn "Đang mượn"   | Hiển thị nút "Trả sách" trên giao diện          |
|                           | Đã trả sách       | Phiếu mượn "Đã trả"      | Ẩn nút "Trả sách", chặn thao tác trùng lặp      |
| Thao tác xác nhận từ user | Xác nhận Trả      | Chọn nút "Xác nhận"      | Hệ thống thực hiện trả sách thành công          |
|                           | Hủy thao tác      | Chọn nút "Hủy"           | Hộp thoại đóng lại, giữ nguyên trạng thái phiếu |

| Đặc tính (Characteristic) | Phân vùng (Block) | Giá trị đại diện (Value)      | Kết quả mong đợi                                       |
| ------------------------- | ----------------- | ----------------------------- | ------------------------------------------------------ |
| Quyền (Role) người dùng   | Thủ thư           | librarian@library.com         | Hiển thị tính năng và cho phép quét quá hạn            |
|                           | Thành viên thường | Tài khoản thành viên          | Ẩn nút tính năng, chặn URL trực tiếp                   |
| Dữ liệu phiếu quá hạn     | Có phiếu quá hạn  | Tồn tại phiếu quá hạn (BR001) | Quét tự động, cập nhật trạng thái phiếu sang "Quá hạn" |
|                           | Không có phiếu    | Hệ thống sạch dữ liệu         | Thông báo hệ thống hiển thị "0 phiếu quá hạn"          |

| Giao diện phía Thành viên | Xem phiếu quá hạn | Phiếu mượn bị quá hạn         | Trạng thái hiển thị nổi bật (màu đỏ, chữ đậm)          |

### IDM — Tra cứu phiếu mượn (REQ-08)

| Đặc tính (Characteristic) | Phân vùng (Block)   | Giá trị đại diện (Value)     | Kết quả mong đợi                                          |

| Tính hợp lệ tài khoản?    | Đúng thông tin      | Các tài khoản trong hệ thống | Cho phép đăng nhập thành công để tra cứu dữ liệu          |
